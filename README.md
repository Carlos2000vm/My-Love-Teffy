# My Love Teffy

Una página web romántica creada con Next.js.

## Despliegue

Este proyecto se despliega automáticamente a GitHub Pages cuando se hace push a la rama main.

### Desarrollo local

1. Instala las dependencias:
   ```bash
   pnpm install
   ```

2. Ejecuta el servidor de desarrollo:
   ```bash
   pnpm dev
   ```

3. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

### Build

Para construir el proyecto:
```bash
pnpm build
```

Los archivos estáticos se generan en la carpeta `out/`.
